<h2 id="Lua-介绍"><a href="#Lua-介绍" class="headerlink" title="Lua 介绍"></a>Lua 介绍</h2><p>Lua 是一种轻量小巧的脚本语言，用标准C语言编写并以源代码形式开放。<br/>其设计目的是为了嵌入应用程序中，从而为应用程序提供灵活的扩展和定制功能。</p>

<h4 id="for-循环"><a href="#for-循环" class="headerlink" title="for 循环"></a>for 循环</h4><figure class="highlight plain"><figcaption><span>[基本语法]</span></figcaption><table><tbody><tr><td class="gutter"><pre><span class="line">1</span><br/><span class="line">2</span><br/><span class="line">3</span><br/></pre></td><td class="code"><pre><span class="line">for var=exp1,exp2,exp3 do  </span><br/><span class="line">    &lt;执行体&gt;  </span><br/><span class="line">end </span><br/></pre></td></tr></tbody></table></figure>
<p>var 从 exp1 变化到 exp2，每次变化以 exp3 为步长递增 var，并执行一次 “执行体”。exp3 是可选的，如果不指定，默认为1。<br/></p><figure class="highlight plain"><figcaption><span>[示例]</span></figcaption><table><tbody><tr><td class="gutter"><pre><span class="line">1</span><br/><span class="line">2</span><br/><span class="line">3</span><br/><span class="line">4</span><br/><span class="line">5</span><br/><span class="line">6</span><br/><span class="line">7</span><br/><span class="line">8</span><br/><span class="line">9</span><br/><span class="line">10</span><br/></pre></td><td class="code"><pre><span class="line">&gt; my_method = {}</span><br/><span class="line">&gt; function my_method.print2()</span><br/><span class="line">&gt;&gt; for i=1,2,1</span><br/><span class="line">&gt;&gt; do</span><br/><span class="line">&gt;&gt; print(&#34;haahah&#34;)</span><br/><span class="line">&gt;&gt; end</span><br/><span class="line">&gt;&gt; end</span><br/><span class="line">&gt; my_method.print2()</span><br/><span class="line">haahah</span><br/><span class="line">haahah</span><br/></pre></td></tr></tbody></table></figure><p></p>
<h4 id="repeat…until"><a href="#repeat…until" class="headerlink" title="repeat…until"></a>repeat…until</h4><figure class="highlight plain"><figcaption><span>[基本语法]</span></figcaption><table><tbody><tr><td class="gutter"><pre><span class="line">1</span><br/><span class="line">2</span><br/><span class="line">3</span><br/></pre></td><td class="code"><pre><span class="line">repeat</span><br/><span class="line">   statements</span><br/><span class="line">until( condition )</span><br/></pre></td></tr></tbody></table></figure>
<p>循环条件判断语句（condition）在循环体末尾部分，所以在条件进行判断前循环体都会执行一次。<br/>如果条件判断语句（condition）为 false，循环会重新开始执行，直到条件判断语句（condition）为 true 才会停止执行。<br/></p><figure class="highlight plain"><figcaption><span>[示例]</span></figcaption><table><tbody><tr><td class="gutter"><pre><span class="line">1</span><br/><span class="line">2</span><br/><span class="line">3</span><br/><span class="line">4</span><br/><span class="line">5</span><br/><span class="line">6</span><br/><span class="line">7</span><br/><span class="line">8</span><br/><span class="line">9</span><br/><span class="line">10</span><br/><span class="line">11</span><br/></pre></td><td class="code"><pre><span class="line">&gt; my_method = {}</span><br/><span class="line">&gt; function my_method.print2()</span><br/><span class="line">&gt;&gt; local flag = 1</span><br/><span class="line">&gt;&gt;repeat</span><br/><span class="line">&gt;&gt;   print(&#34;haahah&#34;)</span><br/><span class="line">&gt;&gt;   flag = flag + 1</span><br/><span class="line">&gt;&gt;until(flag &gt; 2)</span><br/><span class="line">&gt;&gt; end</span><br/><span class="line">&gt; my_method.print2()</span><br/><span class="line">haahah</span><br/><span class="line">haahah</span><br/></pre></td></tr></tbody></table></figure><p></p>
