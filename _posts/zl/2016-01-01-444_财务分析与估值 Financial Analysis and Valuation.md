<p>企业的价值评估和资产评估是两个不同的概念。</p>
<p>估值——对企业或股权进行估值的过程。</p>
<p>价值组成/来源：主营业务、非经营性净资产——&gt;企业价值 ——&gt;价值分配：负债、其他资本索取权、普通股权益</p>
<p>两类思路：</p>
<ul>
<li><strong>贴现现金流方法</strong>：通过企业创造的（或者股东能够获得的）未来收益进行估值。</li>
<li><strong>乘数法</strong>：通过与其他同类型企业的比较进行估值。</li>
</ul>
<p>资金的时间价值：今天的一块钱 != 明天的一块钱 ，不同时间获得的钱不能简单相加</p>
<p>贴现：将未来的钱折换成今天的钱<br/>现值=E/(1+r)t </p>
<p><em>r为贴现率</em><br/>企业的价值 = E + E/(1+r) + E/(1+r)² + E/(1+r)³ + …… = E/r</p>
<p>永续年金——自由现金流维持稳定增长/数额</p>
<p>预期收益<br/>债权人 —— 利率<br/>股东 —— 机会成本</p>
<p>风险与收益相伴相随</p>
<p>股东预期收益/要求回报率：用行业平均盈利水平衡量股东要求公司给的回报 —— 权益资本成本</p>
<p><em>加权平均资本成本wacc</em></p>
<p>资本资产定价模型CAPM： E(Re) = Rf + β*（Rm - Rf）<br/>E(Re)权益资本的期望值<br/>Rf无风险收益率<br/>Rm市场收益率<br/>Rm-Rf风险溢价<br/>β反映某个公司与市场的相关性<br/>CAPM模型的含义：获得高收益的唯一途径是承担高风险（系统风险才会补偿，个体风险不会补偿）。</p>
<p>估计股权价值——权益资本成本<br/>估计企业价值——加权平均资本成本wacc</p>
<h1 id="预测报表"><a href="#预测报表" class="headerlink" title="预测报表"></a>预测报表</h1><p>投资活动</p>
<ul>
<li>step1：营业收入</li>
<li>step2：营业成本，营业税金及附加，营业费用、管理费用</li>
<li>step3：流动资产，固定资产，在建工程，无形资产</li>
<li>step4：流动负债，长期负债，股本、资本公积</li>
</ul>
<p>投资活动</p>
<ul>
<li>step5：长期投资，财务费用、投资收益，营业外收支，所得税、净利润，盈余公积，未分配利润</li>
<li>step5.5：现金，短期借款</li>
<li>step6：现金流量表</li>
</ul>
<p>预测难点：营业收入的预测，长期投资的预测</p>
<p>营业外收支：不具有持续性，预测数额0</p>
<p>现金流量表附表——只针对经营活动现金流<br/>净利润 + （） - （） = 经营活动现金流</p>
<p>利润跟现金流产生差异原因：<br/>1.净利润与经营利润的差异，调整与经营活动无关的收入和费用<br/>2.经营利润与经营现金流的差异</p>
<ul>
<li>影响利润不影响现金流的项目</li>
<li>影响现金流不影响利润的项目</li>
</ul>
<p>step1：净利润——&gt;经营净利润<br/>净利润中不属于经营利润的项目：固定资产报废/处置净损益、公允价值变动损益、投资收益、财务费用<br/>调整方向：增加利润的项目应该减掉，减少利润的项目应该加上<br/>step2：经营净利润——&gt;经营现金流<br/>影响利润但不影响现金流的业务：当期计提的减值损失（扣除转回的）、固定资产折旧、无形资产摊销，上述项目都减少了利润，但不影响现金流<br/>调整方向：加上</p>
<p>与利润无关但影响经营现金流的业务：资产、负债的变化，经营性应收、应付以及存货、递延所得税资产和负债的变动<br/>调整反向：应收款增加“吃掉现金”，应付款增加带来现金；减去资产增加，加上负债增加</p>
<p>敏感性分析：收入增长、新增投资<br/>合理性分析</p>
<p>收入预测本身更多是一个艺术过程。</p>
<h1 id="自由现金流贴现模型估值"><a href="#自由现金流贴现模型估值" class="headerlink" title="自由现金流贴现模型估值"></a>自由现金流贴现模型估值</h1><p>股利贴现模型：普通股权益价值 = PV（股利） —— 很少用，存在循环论证</p>
<p>权益现金流贴现模型——估计整体现金流:普通股权益价值 = PV（自由现金流 + 非经营现金流 - 负债现金流 - 其他资本索取权现金流） —— 很少用</p>
<p>自由现金流贴现模型（DCF）：普通股权益价值 = PV（自由现金流） + 非经营性资产 - 负债 - 其他资本索取权<br/>现金流表 | ** | 自由现金流表</p>
<ul>
<li>| - | -<br/>经营现金流 | <strong> | 自由现金流<br/>投资现金流 | </strong> | 非经营性现金流<br/>融资现金流 | *<em> | 资本现金流<br/>净现金流 | </em>净现金流相等* | 净现金流</li>
</ul>
<p>NOPAT税后营业净利润<br/>营业收入 | 净利润</p>
<ul>
<li>| -<br/>-营业成本 | +资产减值的损失<br/>-营业费用 | -公允价值变动的收益<br/>-管理费用 | -投资收益<br/>-不含利息的税前经营利润 | -营业外收支<br/>-税收 | +少数股东损益<br/>=NOPAT | +税后利息费用（-收入）<br/>*** | =NOPAT</li>
</ul>
<p>NOPAT + 折旧 + 递延所得税增加 - 经营性流动资产增加 - 存货增加 + 经营性流动负债增加 - 资本性支出（固定资产增加+折旧） - 商誉投资（商誉增加+摊销） = FCF(自由现金流)</p>
<p><em>一般预测三/五年，四年/六年之后自由现金流按永续年金计算</em></p>
<p>调整后现值模型（APV）：普通股权益价值 = PV（无杠杆时的自由现金流）+ 杠杆带来的价值 + 非经营性净资产 - 负债 - 其他资本索取权</p>
<p>剩余收益模型（RIM）：普通股权益价值 = 主营业务的账面价值 + PV（主营业务创造的剩余收益） + 非经营性净资产 - 负债 - 其他资本索取权</p>
<h1 id="剩余收益模型估值与EVA估值"><a href="#剩余收益模型估值与EVA估值" class="headerlink" title="剩余收益模型估值与EVA估值"></a>剩余收益模型估值与EVA估值</h1><p>调整后的现值模型APV</p>
<ul>
<li>公司的负债可以减少税收</li>
<li>杠杆的增加会增加公司的偿债风险</li>
</ul>
<p>clean surplus relationship<br/>$B_t$ = $B_t-1$ + $E_t$ - $d_t$ （$B_t$ 公司年末股东权益账面价值，$E_t$ t年公司净利润，$d_t$ 公司在t年的分红）<br/>$B_t$ - $B_t-1$ = $E_t$ - $d_t$ (公司年末年初的股东权益的账面价值差等于今年盈利减去今年分红)<br/>融资——反向分红；把直接计入资本公积项目调到利润表中</p>
<p>$d_t$ = $E_t$ - （$B_t$ - $B_t-1$）</p>
<p>$ROE_t$净资产报酬率<br/>$E_t$ = $ROE_t$ <em> $B_t-1$<br/>$B_t$ /(1+r) = [(1+r)-r] </em> $B_t$ /(1+r) =$B_t$ - $B_t$ * r/(1+r)</p>
<p>竞争优势：技术水平高、成本低、品牌形象好、销售能力强</p>
<p>EVA估值方法<br/>经营性净资产（投资资本）：只包括有息负债和股东权益</p>
<h1 id="乘数法估值"><a href="#乘数法估值" class="headerlink" title="乘数法估值"></a>乘数法估值</h1><p>每股收益（EPS）：净利润/股数<br/>市盈率P/E = 每股股价/每股收益（EPS） ——通常用市盈率来表现一个公司估值水平的高低<br/>市净率 = 每股股价/每股账面的股东权益数额（净资产/股数）</p>
<p>估计价格 = 可比公司的P/E * 目标公司的EPS</p>
<p>可比公司：业务，增长前景，会计方法，盈余构成，资本结构</p>
<p>g（增长率）不可比的处理：<br/>$PEG_COMP$ = $PE_COMP$ / $g_COMP$<br/>$P_TARGET$ = $PEG_COMP$ <em> $g_SNTARGET$ </em> $E_TARGET$</p>
<p><em>PEG只适用于高增长，低增长仍使用PE</em></p>
<p>财务杠杆不可比的处理：<br/>用无杠杆税前市盈率代替原先的市盈率</p>
<p>会计方法不可比的处理：</p>
<ul>
<li>调整会计差异——净利润是会计处理方法最敏感的数据，营业收入会计处理方法最不敏感的数据；</li>
<li>使用PS比率——使用以现金流为基础的比率，现金流本身对会计处理方法不敏感，P/E = 权益总市值 / EBITDA；</li>
</ul>
<p>对亏损公司进行估值：<br/>使用其他的价值驱动因素——企业注册用户的数量，网站点击率等</p>
<p>对多元化企业的估值：<br/>把一个多元化企业拆分成若干个单业务公司进行估值（无杠杆税前市盈率）。</p>
<p>市净率P/B = COMEQUITY / BV = COMEQUITY/E * E/BV</p>
<p><del>估值过程有很大的随意性和主观性。</del></p>
<p><strong>更准确的定性的判断是我们得到更准确定量结果的重要前提。</strong></p>
<h1 id="对企业价值的再认识"><a href="#对企业价值的再认识" class="headerlink" title="对企业价值的再认识"></a>对企业价值的再认识</h1><p>经济利润是公司的实际收益超过预期收益的水平。</p>
<p>价值根本来源于竞争优势，来自于企业比其他公司更有效利用自己资产，从而创造出更多实际回报。</p>
<p>增长：ROE&gt;R，留存的收益才会创造价值<br/>真正驱动价值的增长一定是剩余收益的增长，价值的增长是实际收益超过预期收益的水平的增长</p>
<p>市盈率：复权后价格计算的市盈率 (P+d)/E = 1 + 1/r<br/>P/E决定于当前与未来剩余收益的差异。</p>
<p>市净率：P/B与当前的盈利水平无关，剩余收益的大小决定P/B。</p>
<p>成熟企业：P/E低，P/B高<br/>高增长企业：P/E P/B都高 —— 低估<br/>复苏、重组的企业：P/E高 P/B低<br/>当前盈利不好，又没有增长潜力的公司：P/E P/B都低  —— 高估</p>
