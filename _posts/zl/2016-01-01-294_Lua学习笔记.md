<p>Lua 是一种高性能, 解释型, 面向对象的语句, 广泛用于各种项目的内嵌语言, 如 redis, nginx, scrapy, 愤怒的小鸟, 魔兽世界等等.</p>
<p>本文主要介绍 Lua 的语法.</p>

<h2 id="1-数据类型"><a href="#1-数据类型" class="headerlink" title="1. 数据类型"></a>1. 数据类型</h2><p>lua 是一个动态类型语言,一个变量可以存储类型的值.<br/>Lua 常用数据类型:</p>
<ul>
<li>空(nil):    空类型只包含一个值,即nil . nil表示空, 没有赋值的变量或标的字段都是 nil.</li>
<li>布尔(boolean):     布尔类型包含 True 和 False 两个值.</li>
<li>数字(number):  整数合浮点数是都是使用数字类型存储.</li>
<li>字符串(string):     字符串类型可以存储字符串,且与Redis的键值一样都是二进制安全的.字符串可以使用单引号或双引号表示,两个符号是相同的. 字符串可以包含转义字符,如 ‘n’,’r’ 等.</li>
<li>表(table):        表类型是Lua 语言中唯一的数据结构,既可以当数组,又可以当字典,十分灵活.</li>
<li>函数(function):    函数是Lua中的一等值(first-class value),可以存储在变量中,作为函数的参数或返回结果.</li>
</ul>
<h2 id="2-变量"><a href="#2-变量" class="headerlink" title="2. 变量"></a>2. 变量</h2><p>Lua 变量分为全局变量和局部变量. 全局变量无需声明就可以直接使用,默认值是 nil .</p>
<pre><code>&gt; print(b)

a = 1　　 -- 为全局变量a赋值
a = nil   -- 删除全局变量的方法是将其复制为 nil . 全局变量没有声明与未声明之分,只有非 nil 和 nil 的区别.
print(b)　-- 无需声明即可使用，默认值是nil
</code></pre><p>声明局部变量的方式为 “local 变量名” :</p>
<pre><code>local c　　--声明一个局部变量c，默认值是nil
local d = 1　--声明一个局部变量d并赋值为1
local e, f　--可以同时声明多个局部变量           

* 局部变量的作用域为从声明开始到所在层的语句块的结尾.
</code></pre><p>声明一个存储函数的局部变量的方法为 :</p>
<pre><code>local say_hi = function ()
    print &#39;hi&#39;
end     
</code></pre><p>变量名必须是<strong>非数字开头</strong>,只能包含<strong>字母</strong>,<strong>数字</strong>和<strong>下划线</strong>,<strong>区分大小写</strong>. 变量名不能与Lua的保留关键字相同, 保留关键字如下:</p>
<pre><code>and break do else elseif end false for function if in local nil not or repeat return then true until while 
</code></pre><h2 id="3-注释"><a href="#3-注释" class="headerlink" title="3. 注释"></a>3. 注释</h2><ul>
<li>单行: <code>--</code> 开始, 到行尾结束.</li>
<li>多行:  <code>--[[ ... ]]</code> .</li>
</ul>
<h2 id="4-赋值"><a href="#4-赋值" class="headerlink" title="4. 赋值"></a>4. 赋值</h2><p>多重赋值 : </p>
<pre><code>local a, b = 1, 2　   -- a的值是1，b的值是2
local c, d = 1, 2, 3　-- c的值是1，d的值是2，3被舍弃了
local e, f = 1　　      -- e的值是1，f的值是nil
</code></pre><p>在执行多重赋值时,Lua会先计算所有表达式的值,比如: </p>
<pre><code>local a = {1, 2, 3}
local i = 1
i, a[i] = i + 1, 5      -- i = 2 ; a = {5,2,3} , lua 索引从 1 开始.
</code></pre><p>lua 中的函数也可以返回多个值</p>
<h2 id="5-操作符"><a href="#5-操作符" class="headerlink" title="5. 操作符"></a>5. 操作符</h2><h3 id="5-1-数学操作符"><a href="#5-1-数学操作符" class="headerlink" title="5.1 数学操作符 :"></a>5.1 数学操作符 :</h3><p>常见的+、-、*、/、%（取模）、-（一元操作符，取负）和幂运算符号^。</p>
<p>数学操作符的操作数如果是字符串,则会自动转换为数字.</p>
<pre><code>print(&#39;1&#39; + 1)　　-- 2
print(&#39;10&#39; * 2)　　-- 20          
</code></pre><h3 id="5-2-比较操作符"><a href="#5-2-比较操作符" class="headerlink" title="5.2 比较操作符 :"></a>5.2 比较操作符 :</h3><ul>
<li><code>==</code> : 比较两个操作数的类型和值是否相等</li>
<li><code>~=</code> : 与 == 结果相反</li>
<li><code>&lt;,&gt;,&lt;=,&gt;=</code> : 大于,小于,小于等于,大于等于.</li>
</ul>
<ol>
<li>比较操作符的结果一定是<strong>布尔类型</strong> ;</li>
<li>比较操作符,<strong>不会</strong>对两边的操作数进行<strong>自动类型转换</strong>.</li>
</ol>
<h3 id="5-3-逻辑操作符"><a href="#5-3-逻辑操作符" class="headerlink" title="5.3 逻辑操作符 :"></a>5.3 逻辑操作符 :</h3><ul>
<li><code>not</code> : 根据操作数的真和假返回false 和 true</li>
<li><code>and</code> : a and b, 如果a 是真,则返回 b , 否则返回 a .</li>
<li><code>or</code>  : a or b , 如果a 是假,则返回 a , 否则返回 b .</li>
</ul>
<ol>
<li><p>只要操作数不是 nil 或 false ,逻辑操作符都认为操作数是真. 特别<strong>注意 0 或 空字符串也被当做真</strong>.</p>
</li>
<li><p>Lua 逻辑操作符支持<strong>短路</strong>，也就是说对于 false and foo() ，lua 不会调用foo函数，or 类似。</p>
</li>
</ol>
<h3 id="5-4-连接操作符"><a href="#5-4-连接操作符" class="headerlink" title="5.4 连接操作符."></a>5.4 连接操作符.</h3><p><code>...</code> 用来连接两个字符串. <strong>连接操作符会自动把数字类型的抓换成字符串类型</strong>.</p>
<h3 id="5-5-取长度操作符"><a href="#5-5-取长度操作符" class="headerlink" title="5.5 取长度操作符."></a>5.5 取长度操作符.</h3><p>是lua5.1 新增的操作符, <code>#</code> ,用来获取字符串或表的长度.</p>
<pre><code>&gt; print(#&#39;hello&#39;)  -- 5
</code></pre><h3 id="5-6-运算符的优先级"><a href="#5-6-运算符的优先级" class="headerlink" title="5.6 运算符的优先级:"></a>5.6 运算符的优先级:</h3><pre><code>^
not # -(一元)
* / %

</code></pre><h2 id="6-if-语句"><a href="#6-if-语句" class="headerlink" title="6. if 语句"></a>6. if 语句</h2><p>语法 :</p>
<pre><code>if 条件表达式 then
    语句块
elseif 条件表达式 then
    语句块
else
    语句块
end
</code></pre><p><strong>注意</strong> : </p>
<ol>
<li><p><strong>Lua 中只有 nil 和 false 才是假, 其余值,包括0 和空字符串,都被认为是真值</strong>. </p>
</li>
<li><p>Lua 每个语句都可以 <code>;</code> 结尾 ,但是一般来说编写 Lua 是会省略 <code>;</code> ,</p>
</li>
<li><p>Lua 并不强制要求缩进,所有语句也可以写在一行中, 但为了增强可读性,建议在注意<strong>缩进</strong>.</p>
<pre><code>&gt; a = 1 b = 2 if a then b = 3 else b = 4 end
</code></pre></li>
</ol>
<h2 id="7-循环语句"><a href="#7-循环语句" class="headerlink" title="7. 循环语句"></a>7. 循环语句</h2><h3 id="7-1-while-循环"><a href="#7-1-while-循环" class="headerlink" title="7.1 while 循环"></a>7.1 while 循环</h3><pre><code>while 条件表达式 do
    语句块
end
</code></pre><h3 id="7-2-repeat-循环"><a href="#7-2-repeat-循环" class="headerlink" title="7.2 repeat 循环"></a>7.2 repeat 循环</h3><pre><code>repeat

    语句块
end
</code></pre><h2 id="8-表类型"><a href="#8-表类型" class="headerlink" title="8. 表类型"></a>8. 表类型</h2><p>表是Lua中<strong>唯一的数据结构</strong>,可以理解为关联数组, <strong>任何类型的值(除了空类型)都可以作为表的索引</strong>.</p>
<pre><code>a = {}　　　　        --将变量a赋值为一个空表
a[&#39;field&#39;] = &#39;value&#39;　--将field字段赋值value
print(a.field)　　    --打印内容为&#39;value&#39;，a.field是a[&#39;field&#39;]的语法糖。      


people = {　　　--也可以这样定义
    name = &#39;tom&#39;,
    age = 29
}
</code></pre><p>当索引为整数的时候表和传统的数组一样，例如：</p>
<pre><code>a = {}
a[1] = &#39;Tom&#39;
a[2] = &#39;Jeff&#39;
</code></pre><p>可以写成下面这样：</p>
<pre><code>a = {&#39;Tom&#39;, &#39;Jeff&#39;}
print(a[1])　　　　--打印的内容为&#39;Tom&#39;        
</code></pre><p>可以使用通用形式的<code>for语句</code>遍历数组,例如:</p>
<pre><code>for index,value in ipairs(a) do     -- index 迭代数组a 的索引 ; value 迭代数组a 的值.
    print(index)
    print(value)
end

-- ipairs 是Lua 内置的函数,实现类似迭代器的功能.
</code></pre><p>数字形式的for语句</p>
<pre><code>for i=1,#a do
    print(i)
    print(a[i])
end
</code></pre><p><code>pair</code> : 迭代器,用来遍历非数组的表值.</p>
<pre><code>person = {
    name = &#39;Tom&#39;,
    age = 29
}
for index,value in pairs(person) do
    print(index)
    print(value)
end
</code></pre><p><code>pairs</code> 与 <code>ipairs</code> 的区别在于前者会遍历所有值不为 nil 的索引, 而后者只会从索引 1 开始递增遍历到最后一个值不为 nil 的整数索引.</p>
<h2 id="9-函数"><a href="#9-函数" class="headerlink" title="9. 函数"></a>9. 函数</h2><p>一般形式:</p>
<pre><code>function(参数列表)
    函数体
end
</code></pre><p>可以将函数赋值给一个局部变量, 比如:</p>
<pre><code>    local square = function(num)
        return num*num
    end

** 因为在赋值前声明了局部变量 square, 所以可以在函数内部引用自身(实现递归).
</code></pre><p>函数参数 :</p>
<ol>
<li>如果实参的个数<strong>小于</strong>形参的个数,则没有匹配到的形参的值为 <code>nil</code> . </li>
<li>相对应的,如果实参的个数<strong>大于</strong>形参的个数,则多出的实参会<strong>被忽略</strong>. </li>
<li><p>如果希望捕获多出的参数(即实现可变参数个数),可以让最后一个形参为 <code>...</code> . </p>
<pre><code>local function square(...)
    local argv = {...}
    for i = 1,#argv do
        argv[i] = argv[i] * argv[i]
    end
    return unpack(argv)   -- unpack 函数用来返回 表 中的元素. 相当于return argv[1], argv[2], argv[3]
end
a,b,c = square(1,2,3)
print(a)    -- 1
print(b)    -- 4
print(c)    -- 9
</code></pre></li>
</ol>
<p><strong>在 Lua 中, return 和 break (用于跳出循环) 语句必须是语句块中的最后一条语句, 简单的说在这两条语句之后只能是 end,else 或 until 三者之一</strong>. </p>
<p><strong>如果希望在语句块中间使用这两条语句,可以认为的使用 do 和 end 将其包围</strong>.</p>
<h2 id="10-标准库-http-www-lua-org-manual-5-1-manual-html-5"><a href="#10-标准库-http-www-lua-org-manual-5-1-manual-html-5" class="headerlink" title="10. 标准库 http://www.lua.org/manual/5.1/manual.html#5"></a>10. 标准库 <a href="http://www.lua.org/manual/5.1/manual.html#5" target="_blank" rel="noopener noreferrer">http://www.lua.org/manual/5.1/manual.html#5</a></h2><p>Lua 的标准库中提供了很多使用的函数, 比如 ipairs,pairs,tonumber,tostring,unpack 都属于标准库中的Base库.</p>
<p>Redis 支持大部分Lua标准库,如下所示:</p>
<pre><code>库名      说明

</code></pre></li>
<li><p>转换大小写 </p>
<pre><code>string.upper(string)
string.lower(string)
</code></pre></li>
<li><p>获取子字符串</p>
<p> string.sub() 可以获取一个字符串从索引 start 开始到 end 结束的子字符串,索引从1 开始. 索引也可以是负数, -1 代表最后一个元素 . </p>

</code></pre></li>
</ol>
<h3 id="10-2-Table库-其中大部分函数都需要表的形式是数组形式"><a href="#10-2-Table库-其中大部分函数都需要表的形式是数组形式" class="headerlink" title="10.2 Table库 : 其中大部分函数都需要表的形式是数组形式."></a>10.2 Table库 : 其中大部分函数都需要表的形式是数组形式.</h3><ol>
<li><p>将数组转换为字符串</p>
<p> table.concat(table [,sep [,i [,j]]])</p>
<pre><code># sep : 以 sep 指定的参数分割, 默认为空.
# i , j : 用来限制要转换的表元素的索引范围. 默认分别为 1 和 表的长度. 不支持负索引.
</code></pre><blockquote>
<p>print(table.concat({1,2,3}))     –123<br/>print(table.concat({1,2,3},’,’,2))  –2,3<br/>print(table.concat({1,2,3},’,’,2,2)) –2</p>
</blockquote>
</li>
<li><p>向数组中插入元素</p>
<p> table.insert(table ,[pos,] value)   # 在指定索引位置 pos 插入元素 value, 并将后面的元素顺序后移. 默认 pos 值是数组长度加 1 , 即在数组尾部插入.</p>
<pre><code>&gt; a = {1,2,4}
&gt; table.insert(a,3,3)  # {1,2,3,4}
&gt; table.insert(a,5)    # {1,2,3,4,5}
&gt; print(table.concat(a,&#39;,&#39;))
1,2,3,4,5
</code></pre></li>
<li><p>从数组中弹出一个元素</p>
<p> table.remove(table,[,pos])  # 从指定的索引删除一个元素,并将后面的元素前移,返回删除元素值. 默认 pos 的值是数组的长度,即从数组尾部弹出一个元素.</p>
<pre><code>&gt; table.remove(a)     --{1,2,3,4}
&gt; table.remove(a,1)   --{2,3,4}
&gt; print(table.caoncat(a,&#39;,&#39;))
2,3,4
</code></pre></li>
</ol>
